Project Overview:
Customer churn is a critical challenge for subscription-based businesses. This project focuses on building a predictive machine learning model to identify customers at high risk of canceling their service. By analyzing historical customer demographics, usage patterns, and account information, this model helps businesses transition from reactive to proactive retention strategies.

Key Features:

Data Preprocessing & Cleaning: Handled missing values, encoded categorical variables, and scaled numerical features for optimal model performance.

Exploratory Data Analysis (EDA): Visualized feature distributions and correlations to uncover the primary drivers behind customer attrition.

Predictive Modeling: Built, trained, and evaluated multiple machine learning algorithms (such as Logistic Regression) to find the best performing model.

Model Evaluation: Assessed model performance using metrics beyond standard accuracy, focusing on Precision, Recall, and the ROC-AUC score to ensure false positives and false negatives were properly balanced.

Tech Stack: Python, Pandas, NumPy, Scikit-Learn, Matplotlib/Seaborn.
